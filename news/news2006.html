November 20, 2006 - AfterStep 2.2.4 released!
    Lots of changes:
    • SVG image format support using librsvg
    • implemented menu balloons. When it is enabled in look ballons will be shown for items that have attached comments (in .desktop files ) and Background pixmap previews will show when item is selected.
    • allowed for localized text in menu entries, generated from .desktop files.
    • added FolderReference setting in .include files to facilitate the above localization.
    • major cleanup and improvement of the main menu structure.
    • new look.Smooth

    Get it from here : ftp.afterstep.org/stable/AfterStep-2.2.4.tar.gz.
    Or if you already have 2.1.0 or later installed - you can use smaller upgrade packages : ftp.afterstep.org/stable/AfterStep-2.2.4-noimages.tar.gz.

October 11, 2006 - AfterStep 2.2.3 released!
    This release includes many bugfixes, enhancements and new features, as well as huge modifications to ASXML parsing. GTK colorscheming is now disabled by default. Support is now implemented for parsing module options from look and feel; only WinList does this for now. Major menu contents cleanup... and so much more stuff that you should probably check out the ChangeLog from the ftp server if you're further interested.

    Get it from here : ftp.afterstep.org/stable/AfterStep-2.2.3.tar.gz.
    Or if you already have 2.1.0 or later installed - you can use smaller upgrade packages : ftp.afterstep.org/stable/AfterStep-2.2.3-noimages.tar.gz.

May 24, 2006 - AfterStep 2.2.2 released!
    Fixed a bunch of memory leaks, implemented automatic image reloading on colorscheme change in Wharf, added look.Abi, urgent window handling in WinList, countered transient windows abusing USPosition, mousewheel events in menus, perl bindings for libAfterImage, support for Window Opacity with xorg composite managers... and more!

    Get it from here : ftp.afterstep.org/stable/AfterStep-2.2.2.tar.gz.
    Or if you already have 2.1.0 or later installed - you can use smaller upgrade packages : ftp.afterstep.org/stable/AfterStep-2.2.2-noimages.tar.gz.
